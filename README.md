## Getting Started

### Prerequisites

- Flutter SDK
- Python 3.11+
- Git

### Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/chronos.git
   ```

2. Install Flutter dependencies:
   ```
   cd chronos
   flutter pub get
   ```

3. Set up Python environment:
   ```
   cd flutter
   pip install -r requirements.txt
   ```

4. Run the app:
   ```
   flutter run
   ```
